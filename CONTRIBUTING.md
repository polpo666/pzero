# Contributing to Pzero

## steps

### 1. fork

https://github.com/polpo666/pzero/fork

### 2. clone

```shell
git clone https://github.com/your_username/pzero
```

### 3. checkout branch

```shell
cd pzero

git checkout -b feat/patch-1
```

### 4. push

```shell
git add .
git commit -m "feat(xx): custom message"
git push
```
